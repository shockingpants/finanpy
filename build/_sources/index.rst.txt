Welcome to finanpy's documentation!
===================================
This library is created to provide simple financial tools that everyone can use
to calculate compounded mortgage payments, interest rates etc. I will add more
complex tools to look at common financial indicators as well as derived
indicators to help analyze portfolio risk/reward ratios, diversification,
and projected growth.
